# 🏡 Casa dos Sonhos

> Planejando cada detalhe do seu novo lar.

---

## ✨ Sobre o projeto

O **Casa dos Sonhos** é uma aplicação web desenvolvida para ajudar na organização das compras da primeira casa.

O objetivo é permitir o gerenciamento de móveis, eletrodomésticos, utensílios, decoração e orçamento em um único lugar, de forma simples, elegante e intuitiva.

---

## 🎯 Funcionalidades

- Dashboard
- Lista de compras
- Cadastro de itens
- Categorias
- Controle financeiro
- Wishlist
- Barra de progresso
- Relatórios
- Responsivo
- Modo escuro
